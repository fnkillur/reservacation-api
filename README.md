# reservacation-api
제주도 카페 예약 사이트 - api

ROUTE | METHOD | DESCRIPTION
---|---|---
/api/users | GET | 모든 user 데이터 조회
/api/user/:id | GET | id 값으로 데이터 조회
/api/user | POST | user 데이터 생성
/api/user/:id | PUT | user 데이터 수정
/api/user/:id | DELETE | user 데이터 제거

# 시작하기

* sequalize-cli 설치
```
npm i -g sequalize-cli
```
